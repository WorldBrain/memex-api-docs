## REST quickstart

1. Get a Supabase access token for the user.
2. Call `search-content` or `save-content-by-url`.

```bash
curl -X POST "https://api.memex.garden/search-content" \
  -H "Authorization: Bearer YOUR_SUPABASE_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "vector databases",
    "limit": 10,
    "offset": 0
  }'
```

## MCP quickstart

1. Generate an MCP API key in Memex settings.
2. Use your user ID + MCP API key to call the MCP endpoint.

```bash
curl -X POST "https://mcp.memex.garden/mcp" \
  -H "Content-Type: application/json" \
  -H "x-user-id: YOUR_USER_ID" \
  -H "x-api-key: YOUR_MCP_API_KEY" \
  -d '{
    "jsonrpc": "2.0",
    "id": "call-1",
    "method": "tools/call",
    "params": {
      "name": "save_content_by_url",
      "arguments": {
        "url": "https://example.com/article",
        "tags": ["reading"]
      }
    }
  }'
```

<Note>
  MCP `search_content` deducts query/rerank credits. REST calls do not apply MCP billing by default.
</Note>
